# UX MANIFESTO – Igreja como Corpo Vivo

Este aplicativo não é um sistema.
É uma extensão da igreja.

## 1. Organizado
O usuário sempre sabe:
- Onde está
- O que pode fazer
- O que é responsabilidade dele

Nada é escondido.
Nada é confuso.

## 2. Acolhido
Desde o primeiro acesso:
- Linguagem humana
- Mensagens claras
- Nenhum “erro técnico” exposto ao usuário

O sistema fala como a igreja falaria.

## 3. Orientado
Todo módulo explica:
- Para que serve
- Quando usar
- Quem pode usar

Nenhuma tela vazia sem contexto.

## 4. Conectado
Pessoas se conectam com pessoas:
- Mensagens
- Pedidos
- Agenda
- Ministérios

Nunca apenas dados.

## 5. Responsável
Cada pessoa vê:
- Suas tarefas
- Seus compromissos
- Seu impacto no corpo

Sem microgerenciamento.
Com clareza.

## 6. Parte do Corpo
O app reflete hierarquia saudável:
- Membros
- Líderes
- Ministérios
- Corpo pastoral / presbitério

Todos pertencem.
Todos servem.
