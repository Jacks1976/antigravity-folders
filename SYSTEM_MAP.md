# SYSTEM MAP – Church Management App

## PROPÓSITO CENTRAL
Criar um sistema que faça cada pessoa se sentir:
1. Organizado
2. Acolhido
3. Orientado
4. Conectado
5. Responsável
6. Parte do Corpo
7. Corpo Pastoral / Presbitério integrado

---

## PERFIS DE USUÁRIOS

### Visitante
- Ver avisos públicos
- Ver agenda pública
- Solicitar cadastro

### Membro
- Ver agenda
- Receber avisos
- Enviar pedidos (oração, visita, ajuda)
- Mensagens internas
- Participar de ministérios

### Líder de Ministério
- Gerenciar membros do ministério
- Escalas
- Materiais (PDF, áudio, vídeos)
- Comunicação interna do ministério

### Corpo Pastoral / Presbitério
- Visão geral de tudo
- Comunicação direta com líderes
- Acompanhamento pastoral
- Aprovações estratégicas

### Admin
- Controle total do sistema
- Usuários
- Permissões
- Estrutura do app

## MÓDULOS DO SISTEMA

### Comunicação
- Avisos
- Mensagens privadas
- Notificações

### Pessoas
- Membros
- Ministérios
- Funções
- Status (ativo, visitante, líder)

### Agenda & Eventos
- Eventos públicos
- Eventos internos
- Escalas

### Louvor
- Repertório
- Letras
- Áudios
- Escalas

### Áudio Visual
- Materiais (flyers, thumbs, vídeos)
- Tutoriais (PDFs)

### Administrativo
- Aprovações
- Relatórios
- Configurações

## REGRA DE OURO
Nada será implementado sem:
- Clareza
- Sentido pastoral
- Simplicidade para o usuário final
